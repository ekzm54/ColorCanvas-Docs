# Book 4 — Design System & UI Bible

## Purpose

Book 4 defines the user interface and interaction system for ColorCanvas.

## Design Principles

- Canvas first.
- Apple Pencil first.
- Zero learning curve.
- Safe Color should feel invisible and reliable.
- UI must not distract from artwork.

## Implementation Rule

All SwiftUI code must use Design Tokens. Raw colors, fonts, spacing, radius, elevation, and animation values are forbidden outside token definitions.
