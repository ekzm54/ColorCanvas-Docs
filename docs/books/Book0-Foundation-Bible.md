# Book 0 — Foundation Bible

## Purpose

Book 0 is the project constitution for ColorCanvas. It defines the highest-level principles that all Books, ADRs, RFCs, Sprints, and code must follow.

## Mission

Create the easiest, most stable, and most enjoyable iPad coloring experience.

## Priority Order

```text
User artwork protection
↓
Apple Pencil responsiveness
↓
Safe Color accuracy
↓
Stability
↓
Maintainability
↓
Performance
↓
New features
```

## Non-Negotiable Principles

- Never lose user artwork.
- Auto Save is always on.
- Local First and Offline First.
- Safe Color is a core product differentiator.
- Engine first, UI second.
- Contract first, implementation second.
- Tests are required for completion.
- AI must not invent features outside scope.

## AI Startup Sequence

1. Read Book 0.
2. Read relevant ADR.
3. Read current RFC.
4. Read current Sprint.
5. Implement only the active task.
